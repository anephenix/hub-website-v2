---
title: Install
description: Instructions for installing Hub
---

You can install Hub via [NPM](https://npmjs.com):

```shell
npm i @anephenix/hub
```

You will want to do this for both the project where you run the server, as well as for any other projects where you want the client to run as well (such as the frontend for a website or as a server-side client script).

## Deno

To install Hub on Deno, run this line in your terminal:

```shell
deno install npm:@anephenix/hub
```

## Bun

To install Hub on Bun, run this line in your terminal:

```shell
bun install @anephenix/sarus
```
