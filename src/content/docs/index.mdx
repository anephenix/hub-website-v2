## Introducing Hub

Hub is a Realtime framework powered by WebSockets and [Node.js](https://nodejs.org/), and built for [Anephenix](https://anephenix.com).

## Install

```shell
npm i @anephenix/hub
```

## Usage

On the server:

```javascript
// Dependencies
import Hub from '@anephenix/hub';

// Initialize hub to listen on port 4000
const hub = new Hub({ port: 4000 });

// Start listening
hub.listen();
```

On the client (in the browser or in Node.js):

```javascript
import HubClient from '@anephenix/hub/client';

// Create an instance of HubClient
const hubClient = new HubClient({ url: 'ws://localhost:4000' });

// Wait for the client to connect and set client ID
await hubClient.isReady();
```
